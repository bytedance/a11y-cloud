# a11y-web-client

云标签标注工具🔧，支持标注过程高亮、功能实时生效。

[English Version](./README.md)

## 项目运行

```
yarn start

# 如果需要切换为中文环境，请使用如下命令
yarn start:zh
```

## 项目构建

```
yarn build

# 如果需要切换为中文环境，请使用如下命令
yarn build:zh
```

## 数据 DSL

数据格式请参考 [DSL.js](../../DSL.js)

## 注意事项

- 需要同时启用 [a11y-web-socket](../a11y-web-socket/) 项目，才可正常收发数据
